Past Project - Data Mining Amazon.com Reviews

In this project I data mined hundreds of amazon reviews for a bag of parakeet birdseed that had 4.5 out of 5 stars. The purpose was to quickly determine the most common reasons behind each customer's review and why they chose to give the amound of stars they did to this product. This project involved the use of jupyter notebooks, beautiful soup, word cloud, and python programming to webscrape reviews off of the product's page on amazon.com. Statistics were performed to determine the spread of star scores among all customers. Reviews were grouped by star score and analyzed to find the top words used by each review group. Wordcloud was used to visually display these words in order to easily intrepret the most common reasons this product received the stars score given.

Link to my Data Mining project.

Past Project - AWS Cloud Networking

In this project I created a virtual network on Amazon Web Services. I created three Virtual Private Clouds with their own EC2 instances that were able to communicate with each other with public IP addresses over Internet Gateways and with private IP addresses through Transit Gateways that I configured. I then expanded the network by using CloudFormation to add a stack containing a website server, DNS server, and bastion server. I configured the bastion server to act as a VPN gateway for this stack. I setup a VPN Tunnnel and Route53 to resolve DNS requests for the test website. I then setup network monitoring using CloudWatch, configured a VPN down alarm, configured ACL lists, configured Security Groups, and managed permissions for S3 buckets used by EC2 instances inside my network.
